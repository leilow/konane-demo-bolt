## 🚀 Quick Start (for teammates)
1. Clone this repo  
2. Install dependencies  
   ```bash
   npm install
   ```
3. Run dev server  
   ```bash
   npm run dev
   ```
   → open http://localhost:5173

Build for production:  
```bash
npm run build && npm run preview
```
